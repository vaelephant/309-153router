# OptRouter 朋友圈单图物料 v1

> 结构参考 SelectDB：上（价值）→ 中（品牌）→ 下（单按钮 CTA）  
> 按钮 **不直接加微信**，统一跳转着陆页，由页面完成转化。

---

## 一、转化链路

```text
朋友圈单图
  ↓ 点击「了解更多」
https://optrouter.com/ai-gateway?from=wechat_moments
  ↓ 着陆页（手机端）
主转化：扫微信 · 次：留电话 / 免费试用（底栏固定）
```

**归因参数（建议固定）：**

| 渠道       | from 参数           |
|------------|---------------------|
| 朋友圈单图 | `wechat_moments`    |
| 微信群转发 | `wechat_group`      |
| 好友私发   | `wechat_dm`         |
| 线下名片   | `offline_card`      |
