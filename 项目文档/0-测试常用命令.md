# 测试常用命令

## 测试 OPENAI_API_KEY 是否可用（直连 OpenAI，用 curl）

先加载 key（在项目根目录执行）：

```bash
export OPENAI_API_KEY="你的key"
```

然后执行（列出模型 = 校验 key）：

```bash
curl -s -H "Authorization: Bearer $OPENAI_API_KEY" https://api.openai.com/v1/models
```

返回一长串 JSON 即表示 key 有效；若 key 无效会返回 401 等错误 JSON。

---

## 非流式（一次返回完整 JSON，可用 jq 格式化）

请求体里 `"stream": false`（或省略），返回整段 JSON：

```bash
curl -X POST http://localhost:9115/v1/chat/completions \
  -H "Authorization: Bearer sk-mm61wdhk-2fcvs5uq9he" \
  -H "Content-Type: application/json" \
  -d '{"model":"gpt-4o","messages":[{"role":"user","content":"请输出 4 个字"}],"stream":false}' | jq
```

## 流式（只输出生成文字）

### 方式一：grep 提取（快速）

```bash
curl -N -s -X POST http://localhost:9115/v1/chat/completions \
  -H "Authorization: Bearer sk-mm61wdhk-2fcvs5uq9he" \
  -H "Content-Type: application/json" \
  -d '{"model":"gpt-4o","messages":[{"role":"user","content":"请输出 4 个字"}],"stream":true}' \
  | grep -o '"content":"[^"]*"' | sed 's/"content":"//;s/"$//'
```

### 方式二：Python 解析（推荐，支持多字节字符）

```bash
curl -N -s -X POST http://localhost:9115/v1/chat/completions \
  -H "Authorization: Bearer sk-mm61wdhk-2fcvs5uq9he" \
  -H "Content-Type: application/json" \
  -d '{"model":"gpt-4o","messages":[{"role":"user","content":"请输出 4 个字"}],"stream":true}' \
  | python3 -c "
import sys, json
for line in sys.stdin:
    line = line.strip()
    if line.startswith('data: ') and '[DONE]' not in line:
        try:
            c = json.loads(line[6:])['choices'][0]['delta'].get('content','')
            print(c, end='', flush=True)
        except: pass
print()
"
```
