# 通用用户系统功能模块清单

## 一、用户认证模块

### 1. 用户注册
- **手机号注册**：手机号格式验证、唯一性检查
- **密码设置**：最小长度验证（至少6位）
- **邀请码注册**：可选，支持 URL 参数和表单输入
- **注册时处理邀请关系**：自动绑定邀请人-被邀请人关系
- **注册日志记录**：记录 IP、User-Agent 等信息

### 2. 用户登录
- **手机号密码登录**：手机号格式验证
- **密码验证**：使用 bcrypt/argon2 加密存储和验证
- **登录日志记录**：记录 IP、User-Agent、登录时间
- **最后登录时间更新**：每次登录更新用户最后登录时间
- **Token 生成与会话管理**：生成认证 Token，管理用户会话

### 3. 用户信息管理
- **用户基本信息**：手机号、角色、创建时间等
- **密码修改**：支持用户修改密码
- **登录历史查询**：查看用户登录记录

---

## 二、用户统计模块

### 4. 用户数据统计（管理员）
- **今日统计**：
  - 今日登录用户数
  - 今日新注册用户数
- **近 7 日统计**：
  - 近 7 日登录用户数
  - 近 7 日新注册用户数
- **累计统计**：
  - 累计登录次数
- **每日趋势图表**：
  - 每日登录用户数趋势
  - 每日新注册用户数趋势
  - 支持时间范围筛选（7天/30天等）

---

## 三、邀请系统模块

### 5. 邀请码管理
- **生成邀请码**：
  - 支持设置使用次数上限（可设置无限次）
  - 支持设置过期时间（可选）
  - 自动生成唯一邀请码
- **查看我的邀请码列表**：
  - 显示所有创建的邀请码
  - 显示每个邀请码的状态（可用次数、已使用次数、是否过期）
- **邀请码使用记录**：记录每个邀请码的使用情况

### 6. 邀请关系管理
- **注册时绑定邀请关系**：新用户注册时，如果提供了邀请码，自动建立邀请人-被邀请人关系
- **查看我的邀请用户列表**：
  - 显示所有被邀请的用户信息
  - 显示被邀请人注册时间、使用时间
- **邀请关系查询**：支持查询邀请关系链

### 7. 邀请统计
- **个人邀请统计**：
  - 邀请码总数
  - 邀请用户总数
  - 今日邀请数
  - 近 7 日邀请数
  - 奖励总数（如适用）
- **邀请趋势图表**：每日邀请用户数趋势曲线图
- **邀请码使用分布**：各邀请码的使用量占比饼图
- **邀请用户时间分布**：被邀请用户注册时间分布饼图

---

## 四、奖励系统模块（可选）

### 8. 邀请奖励
- **奖励规则配置**：可配置奖励触发条件（如：被邀请人首充后奖励邀请人）
- **奖励记录查询**：查看所有奖励发放记录
- **奖励发放**：支持自动发放和手动发放
- **奖励统计汇总**：统计总奖励金额、奖励次数等

## 核心数据模型

### 数据库表结构

1. **User（用户表）**
   - `id`: UUID，主键
   - `phone`: 手机号，唯一索引
   - `password`: 密码哈希（bcrypt/argon2）
   - `role`: 用户角色（user/admin 等）
   - `createdAt`: 创建时间
   - `updatedAt`: 更新时间

2. **InviteCode（邀请码表）**
   - `id`: UUID，主键
   - `code`: 邀请码字符串，唯一索引
   - `userId`: 创建人 ID，外键关联 User
   - `maxUses`: 最大使用次数（999999 表示无限次）
   - `usedCount`: 已使用次数
   - `expiresAt`: 过期时间（可选）
   - `createdAt`: 创建时间

3. **InviteRelation（邀请关系表）**
   - `id`: UUID，主键
   - `inviterId`: 邀请人 ID，外键关联 User
   - `inviteeId`: 被邀请人 ID，外键关联 User
   - `inviteCodeId`: 使用的邀请码 ID，外键关联 InviteCode
   - `usedAt`: 使用时间（注册时间）

4. **UserLoginLog（登录日志表）**
   - `id`: UUID，主键
   - `userId`: 用户 ID，外键关联 User
   - `phone`: 登录手机号（冗余字段，便于查询）
   - `loginAt`: 登录时间
   - `ipAddress`: IP 地址
   - `userAgent`: User-Agent 信息

5. **InviteRewardRecord（邀请奖励记录表，可选）**
   - `id`: UUID，主键
   - `userId`: 获得奖励的用户 ID，外键关联 User
   - `inviteRelationId`: 关联的邀请关系 ID，外键关联 InviteRelation
   - `rewardType`: 奖励类型
   - `rewardValue`: 奖励金额/数值
   - `grantedAt`: 发放时间

---

## 技术特性

- **密码加密存储**：使用 bcrypt 或 argon2 进行密码哈希
- **Token 认证机制**：基于 Token 的用户认证
- **权限控制**：支持用户角色管理（user/admin 等）
- **日志记录**：完整记录注册、登录、邀请等行为
- **数据统计与可视化**：提供图表展示统计数据
- **响应式设计**：支持移动端适配

---

## 使用说明

本功能模块为**通用用户系统**，可独立复用，不依赖具体业务逻辑。适用于需要用户注册、登录、邀请功能的各类 Web 应用。

### 集成方式

1. **数据库迁移**：按照核心数据模型创建数据库表
2. **API 接口**：实现用户认证、邀请系统相关的 API 接口
3. **前端组件**：实现注册、登录、邀请管理的前端页面
4. **权限守卫**：实现 AuthGuard 组件保护需要登录的页面

### 扩展建议

- 可根据业务需求扩展奖励规则
- 可添加第三方登录（OAuth）
- 可添加手机号验证码登录功能
